# X ツイートマネージャー

> 競合アカウントを分析して、AIがツイートを自動生成するCLIツール

---

## ツール全体の流れ

```
┌─────────────────────────────────────────────────────────────┐
│                    python main.py で起動                     │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
              ┌───────────────────────────┐
              │         メインメニュー        │
              └───────────────────────────┘
                    │         │         │
                    ▼         ▼         ▼
              ┌─────────┐ ┌──────┐ ┌─────────┐
              │ 1. 競合  │ │2. ツ │ │ 3. 保存 │
              │   分析   │ │イート│ │  済み   │
              │         │ │ 生成 │ │  一覧   │
              └─────────┘ └──────┘ └─────────┘
```

## 機能 1｜競合アカウント分析

```
  あなた                X API              Claude AI          保存
    │                     │                    │                │
    │  @ユーザー名を入力    │                    │                │
    │──────────────────►  │                    │                │
    │                     │  ツイートを取得      │                │
    │                     │  (最大100件)        │                │
    │  ツイートデータ       │                    │                │
    │ ◄──────────────────  │                    │                │
    │                     │  ツイートを送信      │                │
    │─────────────────────────────────────────► │                │
    │                     │  分析レポートをJSON  │                │
    │ ◄──────────────────────────────────────── │                │
    │                     │                    │  JSONを保存     │
    │──────────────────────────────────────────────────────────► │
    │                     │                    │                │
    │        分析結果を画面表示                   │                │
```

### 分析で得られる情報

| 項目 | 内容 |
|------|------|
| 投稿頻度 | 1日あたりの投稿ペース |
| トーン | カジュアル / フォーマル / 専門的 など |
| キーワード | よく使う単語トップ10 |
| ハッシュタグ | 頻出タグ一覧 |
| スタイル特徴 | 文体・構成の特徴 |
| 高エンゲージメント | いいね・RTが多い投稿パターン |
| 平均いいね / RT | エンゲージメント指標 |

## 機能 2｜ツイート生成・投稿

```
  あなた             保存済み分析          Claude AI           X
    │                    │                    │                │
    │ テーマを入力        │                    │                │
    │ トーンを選択        │                    │                │
    │ 戦略を選択          │                    │                │
    │                    │                    │                │
    │  分析データを参照？  │                    │                │
    │──────────────────► │                    │                │
    │  分析データ         │                    │                │
    │ ◄────────────────── │                    │                │
    │                    │  テーマ+トーン+分析  │                │
    │─────────────────────────────────────────► │                │
    │                    │   ツイート3案        │                │
    │ ◄──────────────────────────────────────── │                │
    │                    │                    │                │
    │  各案を確認して選択  │                    │                │
    │    ┌───────────┐    │                    │                │
    │    │ post      │──────────────────────────────────────►  │
    │    │ copy      │ → クリップボードにコピー  │                │
    │    │ skip      │ → 次の案へ              │                │
    │    └───────────┘    │                    │                │
```

### 生成戦略の違い

```
  競合を模倣 (similar)          差別化 (differentiated)
  ┌────────────────────┐        ┌────────────────────┐
  │ 競合と似たスタイル  │        │ 競合とは別の視点   │
  │ ・同じトーン        │        │ ・独自のアングル   │
  │ ・似た表現方法      │        │ ・対照的な切り口   │
  │ ・同じキーワード系  │        │ ・差別化した価値観  │
  └────────────────────┘        └────────────────────┘
        競合の人気に乗る                独自性を出す
```

## セットアップ

### 1. 必要なAPIキーを取得

```
X (Twitter) Developer Portal    Anthropic Console
┌──────────────────────────┐   ┌──────────────────┐
│ Bearer Token             │   │ API Key          │
│ API Key                  │   └──────────────────┘
│ API Secret               │
│ Access Token             │
│ Access Token Secret      │
└──────────────────────────┘
※ タイムライン読み取りは Basic tier ($100/月) が必要
```

### 2. 環境構築

```bash
# リポジトリをクローン
git clone https://github.com/ogatasuke/x-tweet-manager.git
cd x-tweet-manager

# 依存ライブラリをインストール
pip install -r requirements.txt

# APIキーを設定
cp .env.example .env
# .env をエディタで開いてキーを入力
```

### 3. .env の記入例

```
X_BEARER_TOKEN=AAA...
X_API_KEY=BBB...
X_API_SECRET=CCC...
X_ACCESS_TOKEN=DDD...
X_ACCESS_TOKEN_SECRET=EEE...
ANTHROPIC_API_KEY=sk-ant-...
```

### 4. 起動

```bash
python main.py
```

## API制限について

| 操作 | 制限 | このツールの対応 |
|------|------|----------------|
| タイムライン読み取り | Basic tier 必須 | 権限エラー時はキャッシュ使用 |
| ツイート投稿 | 月500件まで | 投稿前に確認プロンプト表示 |
| Claude API | トークン課金 | 分析・生成ごとに呼び出し |
