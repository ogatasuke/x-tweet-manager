Overview

3つの主要機能

1

競合アカウント分析

@ユーザー名を入力するだけで、投稿頻度・文体・キーワードをAIが分析。JSON形式で保存。

2

ツイート生成・投稿

テーマとトーンを入力 → AIがツイート3案を生成。投稿またはクリップボードコピー。

3

保存済み分析の一覧

過去の分析結果を一覧表示。ツイート生成時に任意の分析データを参照可能。

Setup

セットアップ

1

APIキーを取得

2つのサービスのAPIキーが必要です。

🐦
X Developer Portal
Bearer Token / API Key+Secret / Access Token+Secret
⚠️ タイムライン読み取りは Basic tier ($100/月) 必須
🤖
Anthropic Console
API Key（sk-ant-...）
2

リポジトリをクローン

# リポジトリをクローン
$ git clone https://github.com/ogatasuke/x-tweet-manager.git
$ cd x-tweet-manager

# 依存ライブラリをインストール
$ pip install -r requirements.txt
3

.env ファイルを作成

$ cp .env.example .env

以下のキーを .env に入力：

X_BEARER_TOKEN=AAA...
X_API_KEY=BBB...
X_API_SECRET=CCC...
X_ACCESS_TOKEN=DDD...
X_ACCESS_TOKEN_SECRET=EEE...
ANTHROPIC_API_KEY=sk-ant-...
4

起動

$ python main.py

メニューが表示されます：

  1. 競合アカウントを分析する
  2. ツイートを生成・投稿する
  3. 保存済み分析を一覧表示
  q. 終了
